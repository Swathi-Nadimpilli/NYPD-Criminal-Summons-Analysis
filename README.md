# NYPD-Criminal-Summons-Analysis
An analysis of law enforcement dynamics using NYC Criminal Summons Data This analysis identifies patterns in crime by analysing different summons, demographics, and geographic details. By leveraging advanced data cleaning techniques and powerful machine learning models like K-Nearest Neighbours and Random Forest, we've achieved remarkable accuracy rates, respectively. The findings allow law enforcement organisations and decision-makers to allocate resources more wisely. To improve public safety by encouraging proactive crime prevention methods, police can put their efforts where they are most needed by identifying locations with higher summons rates.
